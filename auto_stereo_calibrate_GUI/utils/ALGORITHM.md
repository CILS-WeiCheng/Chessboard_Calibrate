# 空間鎖定版智能雙目棋盤格挑選器 (Spatial Locked Stereo Selector) 演算法說明

本文件說明 `stereo_picker.py` 中 `OptimizedStereoChessboardSelector` 類別的核心演算法流程與邏輯概念。

## 1. 核心目標
本演算法旨在從大量的雙目棋盤格影像對中，自動挑選出最有利於相機校正（Stereo Calibration）的影像子集。挑選基準包含：
- **空間覆蓋率 (Spatial Coverage)**：確保棋盤格遍佈影像的 3x3 區域。
- **姿勢多樣性 (Pose Diversity)**：涵蓋不同的傾斜角度與縮放比例。
- **影像品質 (Image Quality)**：排除檢測誤差較大或模糊的影像。
- **誤差最小化 (RMS Optimization)**：透過迭代優化，使最終校正的均方根誤差 (RMS) 達到目標值（如 < 0.5 px）。

## 2. 演算法流程

### 第一階段：特徵分析與評分 (Analysis & Scoring)
針對每一張影像，執行以下步驟：
1. **角點檢測與精化**：使用 `findChessboardCorners` 與 `cornerSubPix` 取得次像素等級的角點坐標。
2. **幾何特徵計算**：
    - **中心坐標**：計算棋盤格在影像中的正規化中心位置。
    - **區域歸屬 (Region)**：將影像劃分為 3x3 的九宮格（ID 0-8），判定棋盤格所處區域。
    - **覆蓋率 (Coverage)**：計算棋盤格佔影像面積的比例。
    - **傾斜角 (Angle)**：計算棋盤格的旋轉角度，增加校正時的幾何约束。
    - **縮放比 (Scale)**：估算棋盤格與影像的大小比例。
3. **靜態投影誤差評估**：利用傳入的相機內參，透過 `solvePnP` 計算該張圖的重投影誤差。若誤差過大（> 10.0 px），則視為無效。
4. **單圖評分 (Score)**：綜合覆蓋率、角度加權與誤差懲罰，計算出該影像的品質分數。

### 第二階段：雙目配對 (Stereo Pairing)
1. 根據檔名將左右相機影像配對。
2. 只有左右兩張圖都成功檢測到角點且誤差在容許範圍內，才保留該配對。
3. **配對分數 (Pair Score)**：取左右兩圖分數的最小值作為該配對的基礎分。

### 第三階段：智能挑選策略 (Selection Strategy)
這是演算法的核心，分為以下三個子步驟：
1. **空間保底選取**：從九個區域（0-8）中，各選出一張分數最高的影像配對，確保視野的全面覆蓋。
2. **多樣性填充 (K-Means Clustering)**：
    - 若空間保底選完後未達目標張數，則將剩餘候選影像根據其「角度」與「縮放比」進行 K-Means 分群。
    - 從各個分群中挑選代表性影像，確保選出的影像在姿勢與尺度上具有高度差異性，而非重複類似的畫面。
3. **迭代誤差優化 (Iterative Optimization)**：
    - 進行初步的雙目校正 (`stereoCalibrate`) 並計算當前 RMS。
    - **尋找最差貢獻者**：計算每對影像對整體誤差的貢獻。
    - **區域鎖定機制 (Spatial Locking)**：
        - 若欲剔除的影像對是其所屬區域的「唯一代表」，則必須從該區域的候選池中尋找替換者，以維持空間覆蓋。
        - 若該區域有多張影像，則允許更靈活的替換。
    - 持續更換高誤差影像，直到 RMS 降低至目標值或達到最大迭代次數。

### 第四階段：結果輸出與報表 (Output & Report)
1. 將選定的影像對重新命名並儲存至指定目錄。
2. 產生 `calibration_selection_report.json`，詳細記錄每張選取圖片的區域、分數與誤差。
3. 繪製 `selection_analysis.png`，視覺化呈現空間分佈（柱狀圖）與角度分佈（直方圖）。

## 3. 邏輯概念總結

| 概念 | 目的 |
| :--- | :--- |
| **3x3 Grid** | 解決校正時邊緣畸變補償不足的問題。 |
| **Static Error Penalty** | 在挑選前即排除明顯異常的影像。 |
| **K-Means Clustering** | 避免模型過度擬合於單一姿勢，提高校正後的泛化能力。 |
| **Iterative Optimization** | 透過動態調整，將最終精度控制在亞像素等級。 |
| **Spatial Locking** | 在追求極限精度時，仍強制維持幾何結構的完整性。 |
